# Top 15 Current Active Java Language Experts

## Oracle Java Champions Program

### Overview
The Oracle Java Champions Program consists of a unique group of skilled Java technologists and community leaders sponsored by Oracle. They are leaders, influencers, and enablers who help grow the size of the Java community. Java Champions are active in many ways such as actively participating in Java projects, engaging with Java User Group communities, speaking at conferences, authoring content, teaching other developers, fostering inclusive participation, and much more.
