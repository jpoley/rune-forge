# Joshua Bloch - Effective Java Authority

## Profile Summary
**Joshua J. Bloch** is a renowned software architect and author, best known for his book "Effective Java" and his fundamental contributions to the Java platform. He is considered one of the most authoritative voices on Java best practices and API design.

## Professional Background

### Current Position (2024-2025)
- **Carnegie Mellon University**: Professor of the Practice, Institute for Software Research
- **Teaching Focus**: Software Engineering coursework and practical programming
- **Research Areas**: API design, software engineering best practices, programming language evolution

### Career Timeline
- **Sun Microsystems (1996-2004)**: Distinguished Engineer, Java Platform Group
- **Google (2004-2012)**: Chief Java Architect, Core Libraries Team
- **Carnegie Mellon (2012-Present)**: Academic leadership and education

## Key Contributions to Java

### Java Collections Framework (1997-1998)
