# Essential Java Podcasts

## Core Java Podcasts

### Java Pub House
**Hosts**: Freddy Guime & Bob Paulin | **Frequency**: Bi-weekly | **Duration**: 60-90 min

The longest-running Java podcast focusing on Java development, frameworks, and community insights.

**Key Topics:**
- Core Java language features and updates
- Framework deep-dives (Spring, Hibernate, etc.)
- Performance optimization and JVM tuning
- Developer tools and IDE tips
- Community interviews and insights
- Conference coverage and summaries

**Why Listen:** Comprehensive Java ecosystem coverage with practical insights from experienced developers.

### A Bootiful Podcast
**Host**: Josh Long (Spring Developer Advocate) | **Frequency**: Weekly | **Duration**: 45-60 min

Spring ecosystem focused podcast with interviews from Java and Spring community leaders.

**Key Topics:**
- Spring Framework ecosystem deep-dives
- Cloud-native Java development
- Reactive programming patterns
- Microservices architecture
- Developer interviews and career stories
- Open source project spotlights

**Why Listen:** Insider perspective on Spring ecosystem and cloud-native Java development.

- OpenJDK and Oracle strategy interviews

## JVM Ecosystem

### Java Off-Heap
**Hosts**: Various JVM community members | **Frequency**: Monthly | **Duration**: 30-45 min

JVM-focused podcast covering performance, internals, and ecosystem developments.

**Key Topics:**
- JVM internals and garbage collection
- Performance profiling and optimization
- Memory management techniques
- JVM language comparisons
- Benchmarking methodologies
- Tool and library evaluations

**Why Listen:** Deep technical content on JVM performance and internals.

### Kotlin Podcast
**Hosts**: Hadi Hariri & various Kotlin team members | **Java Relevance**: High | **Duration**: 30-45 min

While Kotlin-focused, highly relevant for Java developers due to JVM interoperability.

**Java-Relevant Topics:**
- JVM interoperability patterns
- Migration from Java to Kotlin
- Comparative language feature discussions
- Android development considerations
- Spring Framework Kotlin integration

**Why Listen:** Understanding JVM language alternatives and interoperability patterns.
