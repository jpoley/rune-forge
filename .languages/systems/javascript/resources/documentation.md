# Official JavaScript & TypeScript Documentation

## Core Language Specifications

### **ECMAScript Language Specification (ECMA-262)**
- **URL**: [tc39.es/ecma262](https://tc39.es/ecma262/)
- **Authority**: TC39 (Technical Committee 39)
- **Content**: Complete JavaScript language specification
- **Updates**: Continuously updated with latest proposals
- **Why Essential**: The definitive source for JavaScript language behavior
- **Best For**: Understanding precise language semantics and edge cases

### **TC39 Proposals Repository**
- **URL**: [github.com/tc39/proposals](https://github.com/tc39/proposals)
- **Content**: All ECMAScript proposals in various stages
- **Process**: Stage 0 (strawperson) through Stage 4 (finished)
- **Why Essential**: See future JavaScript features before they're standardized
- **Best For**: Staying ahead of JavaScript evolution

### **ECMAScript Compatibility Table**
- **URL**: [kangax.github.io/compat-table/es6](https://kangax.github.io/compat-table/es6/)
- **Content**: Browser and Node.js support for ECMAScript features
- **Updates**: Regular updates across all environments
- **Why Essential**: Critical for understanding feature availability
- **Best For**: Making informed decisions about feature usage
