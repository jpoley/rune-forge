# Essential C# and .NET Podcasts

## Overview
Curated collection of the most valuable podcasts for C# and .NET developers, covering technical deep dives, industry insights, career development, and ecosystem updates.

## Primary .NET-Focused Podcasts

### 1. **.NET Rocks!**
**URL**: https://www.dotnetrocks.com/
**Hosts**: Carl Franklin, Richard Campbell
**Format**: Interview-style, technical discussions
**Episode Length**: 60-75 minutes
**Frequency**: Weekly
**Running Since**: 2002 (one of the longest-running developer podcasts)

**Focus Areas**:
- .NET technology deep dives
- Expert interviews with Microsoft team members
- Industry trends and analysis
- Developer tools and practices
- Career advice and professional development

**Why Listen**: Authoritative .NET content with access to Microsoft insiders, comprehensive coverage of .NET ecosystem evolution.

**Best Episodes**: Annual predictions shows, Microsoft Build coverage, deep dives with .NET team members.

### 2. **The .NET Core Podcast**
**URL**: https://dotnetcore.show/
**Host**: Jamie Taylor
**Format**: Interview and discussion
**Episode Length**: 45-60 minutes
**Frequency**: Bi-weekly

**Focus Areas**:
- Modern .NET development (Core, 5+, 6+)
- Open source .NET projects
- Cross-platform development
- Performance optimization
- Community contributions

**Why Listen**: Focused specifically on modern .NET, excellent guest lineup from .NET community.

**Notable Guests**: Scott Hanselman, Jon Skeet, Julie Lerman, Jimmy Bogard.

### 3. **The Unhandled Exception Podcast**
**URL**: https://unhandledexceptionpodcast.com/
**Host**: Dan Clarke
**Format**: Interview-based with technical focus
**Episode Length**: 45-90 minutes
**Frequency**: Monthly

**Focus Areas**:
- Software development practices
- .NET technology discussions
- Developer career guidance

- Industry expert insights

**Why Listen**: In-depth technical discussions, excellent preparation and questioning.

### 4. **Away from the Keyboard**

## Listening Strategy and Organization

### **Weekly Listening Schedule**
- **Monday**: .NET Rocks! (new episodes)
- **Wednesday**: Hanselminutes (consistent short format)
- **Friday**: The .NET Core Podcast (technical deep dive)
- **Weekend**: Catch up on architecture and career podcasts

### **Subscription Management**
**Recommended Podcast Apps**:
- **Apple Podcasts**: Built-in iPhone integration
- **Spotify**: Cross-platform availability
- **Pocket Casts**: Advanced playlist features
- **Overcast**: Smart speed and voice boost features

### **Learning and Note-Taking**
- Create show note summaries for technical episodes
- Follow up on mentioned tools and technologies
- Join related communities and forums
- Implement discussed techniques in personal projects

### **Conference Season Strategy**
- Prioritize conference recap episodes
- Listen to pre-conference preparation shows
- Follow up with detailed technical sessions
- Connect with speakers on social media

### **Career Development Focus**
- Balance technical and career-focused content
- Take action on career advice episodes
- Network with podcast communities
- Share insights with team members

This comprehensive podcast guide ensures continuous learning and professional development while staying current with C# and .NET ecosystem evolution.
