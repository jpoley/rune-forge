# Go Programming Podcasts & Audio Resources

## Primary Go Podcasts

### Go Time - changelog.com/gotime
- **Hosts**: Mat Ryer, Johnny Boursiquot, and rotating panel
- **Frequency**: Weekly
- **Format**: 45-60 minute discussions
- **Rating**: 5/5 - Essential listening
- **Content Focus**:
  - Language updates and features
  - Community interviews
  - Best practices discussions
  - Tool reviews and comparisons
- **Notable Episodes**:
  - #100: Creating Go with Rob Pike & Robert Griesemer
  - #77: Dependencies and the future with Russ Cox
  - #200: Russ Cox on passing the torch
- **Best For**: Staying current with Go ecosystem

### The Changelog - changelog.com (Go episodes)
- **Hosts**: Adam Stacoviak, Jerod Santo
- **Go Content**: Periodic focus episodes
- **Notable Go Episodes**:
  - Interviews with Go team members
  - Major project announcements
  - Community milestones
- **Best For**: High-level Go ecosystem insights
