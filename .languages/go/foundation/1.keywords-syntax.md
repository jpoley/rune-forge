# Go Keywords and Syntax

## Complete List of Go Keywords (25 total)

### Declaration Keywords
- `const` - Declare constants
- `func` - Declare functions
- `import` - Import packages
- `package` - Declare package name
- `type` - Declare types
- `var` - Declare variables

### Control Flow Keywords
- `break` - Exit from loop or switch
- `case` - Define switch/select cases
- `continue` - Skip to next iteration
- `default` - Default case in switch/select
- `else` - Alternative branch in if statement
- `fallthrough` - Continue to next case in switch
- `for` - Loop construct (only looping keyword)
- `goto` - Jump to labeled statement
- `if` - Conditional statement
- `return` - Return from function
- `select` - Multiplex channel operations
- `switch` - Multi-way branch

### Concurrency Keywords
- `chan` - Channel type
- `go` - Launch goroutine

### Memory and Structure Keywords
- `defer` - Schedule function call for function exit
- `interface` - Define interface type
- `map` - Map type
- `range` - Iterate over slice, map, channel, string
- `struct` - Define structure type

## Syntax Structures

### Package Declaration
```go
package main
```

### Import Statements
```go
import "fmt"
import (
    "fmt"
    "strings"
    "net/http"
)

// Import aliases
import f "fmt"
import _ "database/sql"  // Import for side effects
import . "math"          // Import into current namespace
```

### Function Syntax
```go
// Basic function
func functionName(param1 type1, param2 type2) returnType {
    return value
}

// Multiple return values
func swap(x, y int) (int, int) {
    return y, x
}

// Named return values
func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return  // Naked return
}

// Variadic function
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// Function as first-class citizen
var fn func(int) int = func(x int) int {
    return x * 2
}
```

### Variable Declaration
```go
// Explicit type
var name string = "Go"

// Type inference
var name = "Go"

// Short declaration (inside functions only)
name := "Go"

// Multiple variables
var x, y, z int = 1, 2, 3
a, b, c := 1, 2, 3

// Blank identifier
_, err := someFunction()
```

### Constants
```go
const Pi = 3.14159
const (
    StatusOK = 200
    StatusNotFound = 404
)

// Iota (enumeration)
const (
    Sunday = iota  // 0
    Monday         // 1
    Tuesday        // 2
)
```

### Control Flow Syntax

#### If Statement
```go
if condition {
    // code
}

if x := compute(); x > 0 {
    // can use x here
}

if condition {
    // code
} else if anotherCondition {
    // code
} else {
    // code
}
```

#### For Loop
```go
// Traditional for
for i := 0; i < 10; i++ {
    // code
}

// While-style
for condition {
    // code
}

// Infinite loop
for {
    // code
}

// Range loop
for index, value := range slice {
    // code
}
```

#### Switch Statement
```go
switch expression {
case value1:
    // code
case value2, value3:
    // code
default:
    // code
}

// Type switch
switch v := i.(type) {
case int:
    // v is int
case string:
    // v is string
default:
    // unknown type
}
```

### Type Definitions

#### Struct
```go
type Person struct {
    Name string
    Age  int
}

// Embedded struct
type Employee struct {
    Person
    ID int
}
```

#### Interface
```go
type Writer interface {
    Write([]byte) (int, error)
}

// Empty interface
var x interface{}
```

#### Type Alias
```go
type MyInt int
type StringAlias = string  // True alias (Go 1.9+)
```

### Concurrency Syntax

#### Goroutines
```go
go functionName()
go func() {
    // anonymous function
}()
```

#### Channels
```go
// Create channel
ch := make(chan int)
ch := make(chan int, 100)  // Buffered

// Send
ch <- value

// Receive
value := <-ch

// Close
close(ch)

// Select
select {
case msg1 := <-ch1:
    // handle msg1
case msg2 := <-ch2:
    // handle msg2
case <-time.After(1 * time.Second):
    // timeout
default:
    // non-blocking
}
```

### Defer Statement
```go
defer file.Close()
defer func() {
    // cleanup
}()

// Multiple defers (LIFO)
defer fmt.Println("3")
defer fmt.Println("2")
defer fmt.Println("1")
// Output: 1 2 3
```
