# Jaana Dogan (JBD)

## Overview
Jaana Dogan, also known as JBD, is a highly influential software engineer and former Google Go team member who has made significant contributions to Go's tooling ecosystem and observability infrastructure. She's recognized as one of the leading experts in Go performance, debugging, and production systems.

## Background
- **Role**: Former Google Go team, Principal Engineer at AWS, Founder at Rakyll
- **Location**: San Francisco, CA, USA
- **Years in Go**: 2014-present (10+ years)
- **Known For**: Go tooling, performance, observability, distributed systems

## Key Contributions

### Google Go Team Contributions
- **Go Tooling**: Major contributor to Go's debugging and profiling tools
- **pprof Integration**: Enhanced Go's profiling capabilities
- **trace Package**: Contributed to Go's execution tracing infrastructure
- **Performance Analysis**: Advanced Go performance debugging techniques

### Open Source Projects
- **hey**: HTTP load testing tool
  - URL: https://github.com/rakyll/hey
  - 18k+ stars, widely used for HTTP benchmarking
  - Modern alternative to Apache Bench (ab)

- **boom**: HTTP load testing (predecessor to hey)
  - URL: https://github.com/rakyll/boom
  - Original HTTP load testing tool in Go

- **statik**: Static file embedding for Go
  - URL: https://github.com/rakyll/statik
  - Embeds static files into Go binaries

- **gom**: Dependency management tool (pre-modules era)
  - URL: https://github.com/rakyll/gom
  - Early Go dependency management solution

### Observability and APM
- **OpenCensus**: Co-creator of OpenCensus project
  - Unified stats collection and distributed tracing
  - Later merged into OpenTelemetry
- **Jaeger Integration**: Contributed to Go tracing ecosystem
- **Prometheus Integration**: Go metrics and monitoring patterns
