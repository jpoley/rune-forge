# Mat Ryer

## Overview
Mat Ryer is a prominent Go developer, author, speaker, and open-source contributor. He's the author of "Go Programming Blueprints" and has built numerous popular Go tools and libraries used throughout the community.

## Background
- **Role**: Founder of Machine Box (acquired by Veritone), Senior Engineer at Grafana Labs
- **Location**: UK
- **Years in Go**: 2012-present (11+ years)
- **Known For**: Go tooling, community leadership, practical Go patterns

## Key Contributions

### Open Source Projects
- **Moq**: Code generation tool for creating mock implementations
  - URL: https://github.com/matryer/moq
  - 2k+ stars, widely used for testing in Go

- **Is**: Lightweight testing mini-framework for Go
  - URL: https://github.com/matryer/is
  - Simple assertion library with clear error messages

- **Vice**: Go channels at horizontal scale (message queues)
  - URL: https://github.com/matryer/vice
  - Abstraction layer for message queue systems

- **Silk**: Markdown-based testing for REST APIs
  - URL: https://github.com/matryer/silk
  - Human-readable API testing

### Publications
- **Book**: "Go Programming Blueprints" (Packt Publishing)
  - Practical Go patterns and real-world applications
  - Second edition covers modern Go practices

### Speaking and Education

- **Conferences**: Regular speaker at Go conferences worldwide
- **Workshops**: Conducts Go training and workshops
