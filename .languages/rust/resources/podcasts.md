# Rust Programming Podcasts

## Premier Rust-Focused Podcasts

### 1. Rustacean Station
- **URL**: https://rustacean-station.org/
- **RSS Feed**: https://rustacean-station.org/podcast.rss
- **Description**: Community project podcast exploring the \"weird, wonderful, and wily world of Rust\"
- **Format**: Interview-based episodes with expert guests
- **Release Schedule**: Regular episodes (typically weekly or bi-weekly)
- **Target Audience**: All levels, from beginners to experts
- **Notable Episodes**:
  - \"Rust for Rustaceans by Jon Gjengset\" - Deep dive with the author
  - Rust release highlights and feature discussions
  - Interviews with core team members and community leaders
- **Hosts**: Various community members
- **Platform Availability**: Apple Podcasts, Spotify, Google Podcasts, and direct RSS
- **Community Value**: Central gathering place for Rust community discussions

### 2. New Rustacean: A Podcast About Learning Rust
- **URL**: https://newrustacean.com/
- **Host**: Chris Krycho
- **Status**: Historic (completed series)
- **Description**: Educational podcast documenting one person's journey learning Rust
- **Format**: Tutorial-style episodes with code examples
- **Educational Value**: Excellent for understanding learning progression
- **Target Audience**: Beginners learning Rust
- **Unique Approach**: Learning in public with detailed explanations
- **Legacy**: Important historical resource for Rust education

### 3. Rust in Production
- **Description**: Podcast focusing on real-world Rust usage in production environments
- **Format**: Case studies and interviews with companies using Rust
- **Target Audience**: Developers considering or implementing Rust in production
- **Key Topics**:
  - Production deployment experiences
  - Performance case studies
  - Migration strategies
  - Operational challenges and solutions
- **Value**: Practical insights from real-world implementations

2. Move to **Rustacean Station** for community perspectives
3. Add domain-specific podcasts based on interests

### Intermediate Path
1. **Rustacean Station** for current discussions

3. **Software Engineering Radio** Rust episodes for deep dives
4. Conference recordings for advanced topics

### Advanced Path
